Installation
============

From Source
-----------

.. code-block:: bash

    git clone https://github.com/ryannikolaidis/nllm.git
    cd nllm
    make install

Development Installation
------------------------

.. code-block:: bash

    git clone https://github.com/ryannikolaidis/nllm.git
    cd nllm
    make install-dev

Requirements
------------

* Python 3.12+
