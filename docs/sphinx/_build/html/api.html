Multi-model fan-out wrapper for the llm CLI tool.

Execute the same prompt across multiple AI models concurrently, with structured output and streaming console feedback. Supports all llm flags and options transparently.

Examples

nllm -m gpt-4 -m claude-3-sonnet – “Explain quantum computing” nllm –model-option gpt-4:-o:temperature:0.7 – “Write a haiku” nllm -m gpt-4 –model-option gpt-4:–system:”You are concise” – “Summarize quantum computing” nllm -c my-config.yaml – “What is the capital of France?”
